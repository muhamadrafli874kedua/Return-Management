Panduan Setup

  1. Buka Google Apps Script, buat project baru
  2. Upload file code.gs ke project tersebut
  3. Jalankan fungsi setupSheets() untuk membuat semua sheet otomatis
  4. Isi TELEGRAM_TOKEN dan TELEGRAM_CHAT_ID di sheet KONFIGURASI
  5. Deploy sebagai Web App: Deploy > New deployment > Web App
  6. Set "Who has access" ke Anyone, salin URL
  7. Tempel URL di field di atas dan klik Simpan
